## Environment

- build with Python 3.11.10

```bash
python -m venv venv  # removed via: sudo rm -rf venv
source venv/bin/activate
pip install --upgrade pip wheel setuptools

```

## Start app

```bash
source venv/bin/activate
streamlit run app/app.py
```
